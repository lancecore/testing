Dealing with Default Browser Styling

The problem: The image to the left is off-centre.

The blue outline shows the border of the div element.

The image and its caption are inside a figure element.

The red outline shows the border of the img element.

There are no CSS margin or padding styles applied to the figure or image tags.

The HTML of the div and its contents:

<div>
	<figure>
  		<img src="images/run.gif">
		<figcaption>Caption text</figcaption>
	</figure>
</div>

Using the Web Inspector tool, we can see that default browser styling is being applied to the figure element:

Because the default browser styling comes before our custom CSS in the cascade, we can override these styles in our CSS:

figure {
	margin-left: 0;
	margin-right: 0;
}

The result.

Now we can centre the figure element in our CSS.
